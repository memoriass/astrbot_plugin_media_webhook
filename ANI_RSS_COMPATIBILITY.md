# Ani-RSS 兼容性支持

本插件完全兼容 [Ani-RSS](https://github.com/wushuo894/ani-rss) 项目的 Webhook 通知功能。

## 主要特性

- ✅ **自动格式检测** - 智能识别 Ani-RSS 数据格式
- ✅ **JSON 修复功能** - 自动修复不完整的 JSON 数据
- ✅ **图片支持** - 自动提取和显示动画封面图片

## 配置方法

在 Ani-RSS 中设置 Webhook URL：
```
http://your-bot-server:60071/media-webhook
```
