# 媒体通知 Webhook 插件

为 AstrBot 提供媒体服务器通知功能。原始逻辑来自 @CikeyQi

## 功能特性

- 📱 **智能消息** - 自动格式化，包含剧集信息和封面图片
- 🔄 **智能去重** - 防止重复通知
- 🎯 **数据丰富** - TMDB/BGM.TV 数据补全
- 🖼️ **图片降级** - 自动获取封面图片
- 📦 **批量发送** - 支持消息缓存和合并转发

## 配置

```json
{
  "webhook_port": 60071,
  "group_id": "your-group-id",
  "tmdb_api_key": "your-tmdb-key"
}
```

**媒体服务器设置**：
```
Webhook URL: http://your-bot-server:60071/media-webhook
```

**测试命令**：
```
/webhook test    # 测试功能
/webhook status  # 查看状态
```

## 使用说明

1. 配置插件参数
2. 在媒体服务器中设置 Webhook URL
3. 插件会自动接收并推送通知到指定群组

**发送策略**：
- 消息数量达到阈值：立即合并发送
- 超过时间间隔：自动发送队列中的消息
- 少量消息：使用单独发送避免刷屏
